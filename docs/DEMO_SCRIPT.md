# 🎥 ComplianceIQ Demo Video Script

## BEFORE RECORDING CHECKLIST:
- [ ] Mock mode is ON (toggle in sidebar)
- [ ] Browser zoom at 125% (judges need to see text clearly)
- [ ] Streamlit app running: `streamlit run app/streamlit_app.py`
- [ ] Screen recording software running (OBS or QuickTime)
- [ ] Microphone tested
- [ ] Close all other browser tabs
- [ ] Set screen resolution to 1920x1080

## THE SCRIPT (exact words + actions):

**[0:00-0:10] OPENING**
- **SHOW:** Streamlit app on screen, Title "ComplianceIQ" visible
- **SAY:** "Welcome. This is ComplianceIQ — a 6-agent EU AI Act compliance system built with Microsoft Foundry and Foundry IQ."

**[0:10-0:30] PROBLEM HOOK**
- **SHOW:** Scroll to show the problem description on screen
- **SAY:** "The EU AI Act is fully enforced in 2026. Every company using AI for EU customers must comply or face fines up to 35 million euros. Manual compliance audits cost 50,000 dollars and take 6 weeks. ComplianceIQ does the same audit in 3 minutes. Let me show you."

**[0:30-1:00] UPLOAD SECTION**
- **SHOW:** Click on "Upload & Analyze" tab
- **SAY:** "Our sample AI system is TalentFilter Pro — an automated CV screening system used in HR. This is exactly the kind of system the EU AI Act targets: it makes employment decisions automatically, affecting job applicants across Europe."
- **ACTION:** In mock mode, click "Run Compliance Analysis" button

**[1:00-2:15] AGENT PIPELINE RUNNING**
- **SHOW:** Watch each agent update in real time — hold camera/screen on this
- **SAY:** (as each agent completes)
  - "Agent 1, the Scanner, extracts system characteristics..."
  - "Agent 2, the Gap Analyzer, is now querying our Foundry IQ knowledge base — 4 regulatory sources indexed: EU AI Act, NIST AI RMF, ISO 42001..."
  - "Notice each answer will come with exact article citations from Foundry IQ..."
  - "Agent 3, Risk Scorer, applies EU AI Act Annex III rules..."
  - "Agent 4, Remediation Planner, queries Foundry IQ for best practices..."
  - "Agent 5 writes the reports, Agent 6 verifies every claim against sources..."
  - "And... done."

**[2:15-3:00] RESULTS - RISK DASHBOARD**
- **SHOW:** Click Results tab. Show the 4 metric cards.
- **SAY:** "TalentFilter Pro is classified as HIGH RISK under EU AI Act Annex III, point 4 — employment and recruitment AI. Compliance score: 22%. 4 critical gaps found. Foundry IQ provided 28 citations across all findings."
- **ZOOM:** Show one gap expanded — point to the citation text
- **SAY:** "Every single gap is cited to a specific article in the EU AI Act. This is Foundry IQ working — grounded answers, no hallucination."

**[3:00-4:00] REMEDIATION ROADMAP**
- **SHOW:** Scroll to Remediation section
- **SAY:** "The Remediation Planner generated a 30/60/90 day roadmap. Immediate actions: halt automated rejection — add human review. This is Article 14 — human oversight is mandatory for HIGH RISK systems. 30-day actions: build the risk management system, enable right to explanation. 60-day actions: technical documentation, audit logging. Total estimated effort: 25 working days."

**[4:00-4:40] REPORTS**
- **SHOW:** Click Reports tab. Show all 3 download buttons.
- **SAY:** "Three downloadable reports: Executive summary — board-level, plain English, cites financial risks. Technical report — specific Articles, code changes, architecture requirements. Compliance certificate — formal document ready for regulatory review. All reports include Foundry IQ source citations."
- **ACTION:** Click "Download Executive Report" button

**[4:40-5:00] CLOSE + CALL TO ACTION**
- **SHOW:** Scroll back to top of app. GitHub link visible.
- **SAY:** "ComplianceIQ is fully open source. Deploy in your Azure in one command. Built for the Microsoft Agents League AISF 2026 — Reasoning Agents track. GitHub link in the description. Star the repo. See you on the Discord."
- **SHOW:** Brief flash of the architecture diagram
- **FADE OUT**
